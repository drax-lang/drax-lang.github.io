Welcome to drax

drax is a functional, dynamic programming language built to execute scripts efficiently.

{% highlight drax %}
fun say(name) do
  print('Wellcome, ' ++ name)
end
say('Tom')

#=> prints 'Wellcome, Tom' to STDOUT.
{% endhighlight %}
